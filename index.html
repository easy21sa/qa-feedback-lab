Sample Bug Reports

Login button does not respond

Severity: High

Expected: User should be redirected after clicking login.

Actual: Nothing happens after clicking the button.

Repro Steps:

  1. Open the login page
  2. Enter valid credentials
  3. Click Login
  4. Observe no redirect or loading state

Profile image overlaps text on mobile

Severity: Medium

Expected: Layout should remain readable on smaller screens.

Actual: Image overlaps the username and bio section.

Repro Steps:

  1. Open the profile page on a mobile screen
  2. Scroll to the header section
  3. Observe overlapping elements
